# Transmutador Políglota

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Transmutador Políglota para OpenClaw.

## Qué Soluciona
Traduce sin problemas código entre lenguajes de programación preservando la lógica y la semántica.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales, listos para producción
- Enfoque primero en seguridad
- Soporte para reversión

## Uso
/polyglot-transmogrifier

## Ejemplo
```
Analizar código para detectar problemas de programación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
